# My Dotfiles

This is a very disorganized look at my dotfiles that I hastily put together
and pushed up. Nothing here is really special tbh, the way my terminal looks
in this [video](https://www.youtube.com/watch?v=SgqsnHRfD3Q) is thanks to
[starship](https://starship.rs) for which I've included the config file I use,
it's one of the examples they have on their website.

## Notes:

- My configs will almost certainly not work for you if you just clone the repo
- The `starship.toml` file should go in `~/.config` or something similar. 
- The `.zshrc` file should go in your root directory
- Lots of unnecessary things, feel free to cut them out and have a much neater experience
